# Bash Shell Clone

This repository contains an implementation in Java of a Bash Shell

## Features

The bash clone is capable of command substitution, IO redirection, command sequences and piping

## Commands

Commands that have been implemented are:

mkdir   
rmdir  
ls  
cat  
cut  
grep  
wc  
sed  
find  
datetime  
echo  
head  
tail  
pwd  
whoami
